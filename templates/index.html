Automated Measurement Setup

This wizard will guide you through taking multi-position REW measurements and automatically align them for spatial averaging.

Important: Because REW requires a "Pro License" to automate sweeps via its API, this wizard is designed to work manually with the Free version of REW.

1. Open REW, go to preferences, and start the API Server.
2. Open the Measure dialog.
3. Set Range to 0 to 24000 Hz.
4. Set Level to -12.0 dBFS.
5. Set Timing to Acoustic Timing Reference. (Use C if you have a center channel, otherwise use L).

Position 1

Place the microphone exactly at your Main Listening Position (MLP). This will be the timing anchor.

Measure: L

In REW, select the correct output channel for this speaker and click Start Measuring.
